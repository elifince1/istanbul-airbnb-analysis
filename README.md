# Istanbul Airbnb Analysis

This repository is for my DSA 210 (Introduction to Data Science) term project, Spring 2025–2026.

## What This Project Is About

This project analyzes Istanbul Airbnb listings to understand which listing, host, and review-related features are associated with price and availability patterns. The core research question is:

**Which features of an Airbnb listing are most strongly associated with its price in Istanbul?**

## Data

The primary data comes from [Inside Airbnb](http://insideairbnb.com/get-the-data/) (Istanbul snapshot, September 29, 2025). The dataset includes approximately 30,300 listings described by 79 features. To enrich the analysis, I combine four related files — detailed listings, daily calendar records, individual reviews, and neighbourhood mappings — into a single listing-level analysis dataset. See [reports/proposal.md](reports/proposal.md) for full details.

## What I Want to Examine

- Which listing features (room type, neighbourhood, host attributes, review scores) are most strongly associated with price?
- How do room type, neighbourhood, and host status relate to availability patterns?
- Are there meaningful differences between high-price and lower-price listings across property characteristics?
