# Data Directory

## Planned Data Sources

- **Primary:** [Inside Airbnb – Istanbul](http://insideairbnb.com/get-the-data/) — publicly available listing, review, and calendar data.
- **Enrichment:** Additional contextual data (e.g., neighborhood attributes, tourism indicators) to be determined during the proposal stage.

## Notes

- Raw data files are **not committed** to this repository to keep it lightweight.
- Data acquisition and preparation will be completed in later milestones.
- Once collected, a data dictionary and preprocessing notes will be added here.
